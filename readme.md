# tmux notes

## Introduction

- Terminal Multiplexer = tmux
- essentially runs multiple terminals inside one terminal

## Concept

```txt
Session
    |--> Window
            |--> Pane
```

- Session -> a project
- Window -> a tab
- Pane -> a split inside the Window

## Workflow

1. Start tmux
2. Create a session
3. Use windows for major tasks
4. Split panes inside the windows
5. Detach and attach to the session when needed
